## 소개

사용자가 검색어를 입력하여 관련된 기사 리스트를 보여주는 서비스

## 사용한 기술

- React Hooks, TypeScript, Redux, Styled components

## 구현페이지

### 메인페이지

- 기사 검색 기능
- 기사 정렬 기능 (날짜별, 출처별)
- 기사 클릭 시 새로운 탭 열어 해당 페이지 이동
- 페이지네이션

### 로그인 페이지

- 유저 정보 확인

### 즐겨찾기 페이지

- 로그인 여부에 따라 접근 권한 부여
- redux persist 사용해 즐겨찾기 목록 유지
- 즐겨찾기 추가, 삭제 기능
- 기사 내용 수정 기능
